# User Context

Per-request user context folders generated by `scripts/prepare_contexts.py`.

Each folder is named by `request_id` and contains:

| File | Content |
|------|---------|
| `profile.md` | User embedding summary, pool size, engagement rate |
| `engagement.md` | Engagement history, similarity distribution, top engaged ads |
| `interest_clusters.md` | K-means clusters of positively-engaged ad embeddings |
| `context.md` | Request context: task description, similarity landscape |

## Generating

```bash
python3 scripts/setup_data.py          # full pipeline (checks existing data)
python3 scripts/prepare_contexts.py \  # standalone
  --data-dir data/local/model/split \
  --output-dir user/
```

## Note

Generated `.md` files are gitignored. Only this README is tracked.
