# Ads Pool Understanding

Semantic summary of the candidate ad pool, generated by `ads_pool/refresh.py`.

The agent reads these files before calling any retrieval tools to understand
what kinds of ads are in the pool.

| File | Content |
|------|---------|
| `pool_overview.md` | Pool size, positive rate distribution, embedding statistics |
| `catalog.md` | Per-ad summary: ID, similarity stats, engagement, cluster membership |
| `semantic_clusters.md` | HSNN-style hierarchical clusters with characteristics |

## Generating

```bash
python3 scripts/setup_data.py          # full pipeline (checks existing data)
python3 ads_pool/refresh.py \          # standalone
  --data-dir data/local/model/split
```

## Note

Generated `.md` files are gitignored. Only this README and `refresh.py` are tracked.
