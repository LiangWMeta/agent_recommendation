3. What Changed: V2 → V3

V2: Precision-Focused

Weighted Rank Fusion

score = Σ weight / (rank + 1)

Standard reciprocal rank — heavily favors top-ranked ads from each route. Ads ranked #50 get 1/50th the weight of #1.

Consensus bonus: +1.0 for 3+ routes

Output: 200 ads

Limitation: Over-concentrates on each route's top picks. Positives ranked #80-150 in any route get negligible scores.

V3: Recall-Focused

Flattened Rank Curve

score = Σ weight / (rank + 60)

Key change: +60 instead of +1. Flattens the rank curve. Ad at rank #1 scores weight/61, rank #150 scores weight/210. Ratio is only 3.4x (vs 150x with +1). This gives tail ads a fair chance.

Stronger bonuses: top_positive → +5.0 (was +2.0), 3+ routes → +2.0, 2 routes → +1.0, high-engagement cluster → +1.0

Output: 300 ads (was 200)

Why it works: Positives are spread across all rank positions in each route. The flattened curve ensures positives at rank #100 still contribute meaningful scores, especially when they appear in multiple routes.
